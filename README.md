### flutter_reader

一个简单的 flutter 入门项目，如果你想学 Flutter 而又不知道写什么，用什么接口。我想这个项目能够帮助你。

### 用到的技术

入门级的 Bloc 请求接口数据,入门级的 Redux 控制主题颜色。
